# heart-health-recipes
אנחנו תלמידי מגמת רפואה, וכחלק מעבודת הגמר הקמנו את האתר במטרה לסייע למטופלים עם אי ספיקת לב לנהל אורח חיים בריא וטעים תוך שמירה על הגבלת נתרן ונוזלים. כאן תמצאו מתכונים מותאמים, דלי מלח ועשירים בטעם, שיעזרו לכם לשמור על הלב דרך המטבח. בחרו קטגוריה וגלו מנות בריאות ופשוטות להכנה.
